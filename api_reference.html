  • scm_project_name (str) – the name of the scm project
  • namespace (str) – the namespace in which the scm project resides
  • scm_user (str) – the scm user to connect as, e.g. git
  • scm_host (str) – the hostname of the scm server hosting the git repository
  • scons_script (str) – the name of the SCons script to use, e.g. zupy_packager

  • namespace (str) – The gitlab namespace under which the project resides

  • tag_push (bool) – Whether or not the webhook is triggered with a tag push event
  • issues (bool) – Whether or not the webhook is triggered when an issue is created
  • merge_requests (bool) – Whether or not the webhook is triggered with a merge request is created
